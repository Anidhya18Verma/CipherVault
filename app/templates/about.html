About CipherVault

Built for privacy.
Engineered for trust.

CipherVault is a university project demonstrating secure backend design: JWT authentication, Fernet encryption, and a clean REST API – all wrapped in a production-ready Flask application.

🔐

Zero Plaintext Storage

Every note is encrypted with Fernet (AES-128-CBC + HMAC-SHA256) before touching the database. Raw content never persists.

🎫

Stateless JWT Auth

No server-side sessions. Every request is verified via signed JSON Web Tokens, making the system horizontally scalable.

🛡️

Role-Based Access

Regular users manage their own vault. Admins can view all users and notes through a hidden panel inaccessible from navigation.

🚀

Render-Ready

Configured for deployment on Render with Gunicorn, environment variables, and optional PostgreSQL support.
