Features

Everything you need.

CipherVault is lean by design. Every feature exists to protect your data or make managing it easier.

🔒

Fernet Encryption

Every note is encrypted using Fernet symmetric encryption before storage. Fernet uses AES-128 in CBC mode with HMAC-SHA256 for authentication. Your notes are unreadable without the server key.

🎫

JWT Authentication

Login once, receive a signed token. All API calls verified via Bearer token — no cookies, no sessions.

🔑

Secure Passwords

Passwords are hashed with Werkzeug's PBKDF2-SHA256 before storage. Plaintext never persists anywhere.

⭐

Favorites

Mark important notes as favorites for quick access. Toggle with one tap from your dashboard.

🗂️

Note Management

Create, view, and delete notes. Filter by favorites. All operations require a valid JWT token.

👑

Hidden Admin Panel

An admin panel accessible only via direct URL (/admin-panel) – never exposed in navigation. Admins can view all users and notes system-wide.

🚀

REST API

Full REST API with JSON responses. Every endpoint is documented and returns proper HTTP status codes.

🌐

Render Deployment

Gunicorn-powered, environment-variable driven. Deploy to Render in minutes with PostgreSQL support.
